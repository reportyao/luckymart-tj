## 🔧 使用建议

### 基础使用
```tsx

<AdminPanel />
```

### 定制化使用
```tsx

<AdminPanel 
  title="运营管理面板"
  hideSystemStatus={true}
  customMenuItems={customMenus}
  customQuickActions={customActions}

```

```tsx

  );
}
```
